## О проекте

Интерактивный электронный курс «Язык программирования Python» — статический веб-сайт, разработанный без фреймворков и серверной части. Работает прямо в браузере.

**Технологии:** HTML5 · CSS3 · JavaScript (Vanilla) · Google Fonts

**Возможности:**
- 7 учебных модулей от основ до практики
- Подсветка синтаксиса Python во всех блоках кода
- Кнопка «Копировать» у каждого блока кода
- Интерактивные блоки «Попробуй сам»
- Адаптивный дизайн (мобильные, планшеты, ПК)
- Плавные анимации при прокрутке
- Боковая панель навигации + мобильное меню

## Как запустить у себя

### Способ 1 — Просто открыть файл (самый простой)

1. Нажми зелёную кнопку **Code** → **Download ZIP**
2. Распакуй архив в любую папку
3. Открой файл `index.html` двойным кликом — сайт откроется в браузере

> Никакой установки не требуется. Работает в Chrome, Firefox, Edge, Safari.

---

### Способ 2 — Через Git (для разработчиков)

**Шаг 1.** Убедись, что установлен [Git](https://git-scm.com/downloads)

**Шаг 2.** Клонируй репозиторий:
```bash
git clone https://github.com/Amanch1ik/Python-course-web-site.git
```

**Шаг 3.** Перейди в папку проекта:
```bash
cd Python-course-web-site
```

**Шаг 4.** Открой `index.html` в браузере:
- **Windows:** дважды кликни на `index.html`
- **macOS:** `open index.html`
- **Linux:** `xdg-open index.html`

---

### Способ 3 — Через Live Server (VS Code)

Если хочешь видеть изменения сразу без перезагрузки страницы:

1. Установи [VS Code](https://code.visualstudio.com/)
2. Установи расширение **Live Server** (Ritwick Dey)
3. Открой папку проекта в VS Code
4. Нажми правой кнопкой на `index.html` → **Open with Live Server**
5. Сайт откроется по адресу `http://127.0.0.1:5500`

---

### Способ 4 — Через Python HTTP-сервер

Если установлен Python:

```bash
# Python 3
python -m http.server 8000
```

Открой браузер и перейди по адресу: `http://localhost:8000`
