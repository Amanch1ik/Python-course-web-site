Основы Python

Переменные, типы данных, операторы, строки, ввод и вывод — фундамент любой Python-программы.

Переменные

Переменная — это именованная ячейка памяти, в которой хранится значение. В Python не нужно объявлять тип переменной заранее — он определяется автоматически.

Python 
имя = "Нурмухаммед"       # строка
возраст = 20               # целое число
средний_балл = 4.85        # дробное число
студент = True             # булево значение

print(имя, возраст)         # Нурмухаммед 20

Правила именования переменных

  • Только буквы, цифры и знак подчёркивания _
  • Не может начинаться с цифры
  • Регистр имеет значение: name и Name — разные переменные
  • Нельзя использовать ключевые слова Python (if, for, while и др.)
  • По стандарту PEP 8 — используйте snake_case: средний_балл, имя_пользователя
Множественное присваивание: Python позволяет присвоить значения нескольким переменным в одну строку.
Python 
# Несколько переменных в одну строку
x, y, z = 1, 2, 3
print(x, y, z)  # 1 2 3

# Одно значение — нескольким переменным
a = b = c = 0
print(a, b, c)  # 0 0 0

Типы данных

Python автоматически определяет тип значения. Узнать тип переменной можно функцией type().

Строкаstr"Привет"Текстовые данные в кавычках

НичегоNoneTypeNoneОтсутствие значения
Python 
a = 10
b = 3.14
c = "Python"
d = True

print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>

Преобразование типов

Python 
число_строка = "42"
число = int(число_строка)   # "42" → 42
дробь = float(число)        # 42 → 42.0
обратно = str(число)         # 42 → "42"

print(число + 8)              # 50

Операторы

Арифметические операторы

Python 
a, b = 10, 3

print(a + b)   # 13  — сложение
print(a - b)   # 7   — вычитание
print(a * b)   # 30  — умножение
print(a / b)   # 3.33 — деление
print(a // b)  # 3   — целочисленное деление
print(a % b)   # 1   — остаток от деления
print(a ** b)  # 1000 — возведение в степень

Операторы сравнения

Python 
print(5 == 5)   # True  — равно
print(5 != 3)   # True  — не равно
print(5 > 3)    # True  — больше
print(5 < 3)    # False — меньше
print(5 >= 5)   # True  — больше или равно
print(5 <= 4)   # False — меньше или равно

Логические операторы

Python 
x = 7

print(x > 5 and x < 10)   # True  — оба условия верны
print(x < 5 or x > 6)    # True  — хотя бы одно верно
print(not (x > 5))          # False — инверсия

Строки

Строки (str) — один из самых часто используемых типов данных. Строки можно заключать в одинарные или двойные кавычки.

Python 
имя = "Нурмухаммед"
специальность = 'Бизнес-информатика'

# Конкатенация (склеивание)
print(имя + " — " + специальность)

# f-строки (удобный способ вставки переменных)
группа = "БИ-3-22"
print(f"Студент {имя}, группа {группа}")

# Длина строки
print(len(имя))  # 11

# Методы строк
print(имя.upper())       # НУРМУХАММЕД
print(имя.lower())       # нурмухаммед
print(имя.replace("е", "Е"))  # НурмухаммЕд

Ввод и вывод

Функция print()

Python 
print("Привет!")                        # Привет!
print("a =", 42)                       # a = 42
print("x", "y", "z", sep="-")         # x-y-z
print("Строка 1", end=" | ")
print("Строка 2")                       # Строка 1 | Строка 2

Функция input()

Python 
имя = input("Введите ваше имя: ")
print(f"Привет, {имя}!")

# input() всегда возвращает строку
# Для числа нужно конвертировать:
возраст = int(input("Ваш возраст: "))
print(f"Через 5 лет вам будет {возраст + 5}")
