Модуль 04

Условия и циклы

Управление потоком выполнения программы — ветвления if/elif/else и циклы for/while.

Условный оператор if

Условный оператор if позволяет выполнять разные блоки кода в зависимости от условия. Блок кода внутри if выполняется только тогда, когда условие истинно (True).

Важно: в Python блоки кода выделяются отступами (4 пробела или 1 Tab). Нет фигурных скобок — только отступы.

Python 
температура = 25

if температура > 20:
    print("На улице тепло")   # выполнится

if температура > 35:
    print("Очень жарко!")     # не выполнится

Операторы elif и else

else выполняется, если условие if ложно. elif («else if») проверяет дополнительное условие.

Python 
оценка = 85

if оценка >= 90:
    print("Отлично")
elif оценка >= 75:
    print("Хорошо")         # выполнится это
elif оценка >= 60:
    print("Удовлетворительно")
else:
    print("Неудовлетворительно")
Тернарный оператор: короткая форма if/else в одну строку.
Python 
возраст = 20
статус = "совершеннолетний" if возраст >= 18 else "несовершеннолетний"
print(статус)  # совершеннолетний

Цикл for

Цикл for перебирает элементы последовательности (строки, списка, диапазона). На каждой итерации переменная цикла принимает очередное значение.

Python 
# Перебор списка
фрукты = ["яблоко", "банан", "апельсин"]
for фрукт in фрукты:
    print(фрукт)
# яблоко
# банан
# апельсин

# Перебор строки
for буква in "Python":
    print(буква, end="-")
# P-y-t-h-o-n-

Функция range()

range() генерирует последовательность чисел. Часто используется вместе с for для фиксированного количества повторений.

Python 
# range(стоп) — от 0 до стоп-1
for i in range(5):
    print(i)          # 0 1 2 3 4

# range(старт, стоп)
for i in range(1, 6):
    print(i)          # 1 2 3 4 5

# range(старт, стоп, шаг)
for i in range(0, 11, 2):
    print(i)          # 0 2 4 6 8 10

Цикл while

Цикл while повторяет блок кода, пока условие остаётся истинным. Используйте его, когда заранее неизвестно количество итераций.

Python 
счётчик = 0

while счётчик < 5:
    print(f"Итерация {счётчик}")
    счётчик += 1

# Итерация 0
# Итерация 1
# ...до Итерация 4
Бесконечный цикл: если условие while никогда не станет ложным, программа зависнет. Убедитесь, что переменная условия изменяется в теле цикла.

Операторы break и continue

break — выход из цикла

Python 
for i in range(10):
    if i == 5:
        break       # остановить цикл при i=5
    print(i)
# Выведет: 0 1 2 3 4

continue — пропустить итерацию

Python 
for i in range(6):
    if i % 2 == 0:
        continue    # пропустить чётные числа
    print(i)
# Выведет: 1 3 5
