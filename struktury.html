Структуры данных

Списки, кортежи, словари и множества — основные контейнеры для хранения коллекций данных в Python.

Список — list

Список (list) — упорядоченная изменяемая коллекция элементов. Самая часто используемая структура данных в Python. Элементы могут быть любого типа.

Python 
# Создание списка
студенты = ["Нурмухаммед", "Айгерим", "Даниял"]
оценки = [85, 92, 78, 95]
смешанный = [1, "два", 3.0, True]

# Доступ по индексу (начинается с 0)
print(студенты[0])   # Нурмухаммед
print(студенты[-1])  # Даниял (последний)

# Изменение элемента
студенты[1] = "Акылай"

# Добавление элементов
студенты.append("Бегимай")     # в конец
студенты.insert(1, "Зарина")   # на позицию 1

# Удаление
студенты.remove("Даниял")      # по значению
студенты.pop()                 # последний элемент

# Длина списка
print(len(студенты))

Срезы (slicing)

Python 
числа = [0, 1, 2, 3, 4, 5]

print(числа[1:4])    # [1, 2, 3]
print(числа[:3])     # [0, 1, 2]
print(числа[3:])     # [3, 4, 5]
print(числа[::-1])   # [5, 4, 3, 2, 1, 0] — реверс

Кортеж — tuple

Кортеж (tuple) — упорядоченная неизменяемая коллекция. Используется для данных, которые не должны меняться (координаты, RGB-цвета, конфигурации).

Python 
координаты = (42.87, 74.59)       # Бишкек
цвет_python = (55, 118, 171)       # RGB синий

print(координаты[0])                 # 42.87

# Кортежи нельзя изменять:
# координаты[0] = 0  — это вызовет ошибку!

# Распаковка кортежа
широта, долгота = координаты
print(f"Широта: {широта}, Долгота: {долгота}")

Словарь — dict

Словарь (dict) — коллекция пар «ключ: значение». Ключи уникальны и неизменяемы. Очень удобен для хранения структурированных данных.

Python 
студент = {
    "имя": "Нурмухаммед",
    "группа": "БИ-3-22",
    "средний_балл": 4.7,
    "активен": True
}

# Доступ по ключу
print(студент["имя"])           # Нурмухаммед
print(студент.get("группа"))    # БИ-3-22

# Добавление/обновление
студент["курс"] = 3
студент["средний_балл"] = 4.9

# Перебор словаря
for ключ, значение in студент.items():
    print(f"{ключ}: {значение}")

# Проверка наличия ключа
print("имя" in студент)           # True

Множество — set

Множество (set) — неупорядоченная коллекция уникальных элементов. Дубликаты автоматически удаляются. Поддерживает математические операции: объединение, пересечение, разность.

Python 
языки_а = {"Python", "Java", "C++"}
языки_б = {"Python", "JavaScript", "Go"}

# Объединение
print(языки_а | языки_б)

# Пересечение (общие)
print(языки_а & языки_б)    # {"Python"}

# Разность (только в А)
print(языки_а - языки_б)    # {"Java", "C++"}

# Удаление дубликатов из списка
с_дублями = [1, 2, 2, 3, 3, 3]
без_дублей = list(set(с_дублями))
print(без_дублей)             # [1, 2, 3]
