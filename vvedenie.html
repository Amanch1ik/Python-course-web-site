Введение в Python

Познакомьтесь с языком программирования Python — узнайте его историю, особенности и области применения.

Что такое Python

Python — это высокоуровневый язык программирования общего назначения с динамической типизацией и автоматическим управлением памятью. Он разработан с упором на читаемость кода, поэтому программы на Python выглядят почти как обычный текст.

Python является интерпретируемым языком: это означает, что код выполняется строка за строкой без предварительной компиляции в машинный код. Это делает разработку быстрой и удобной.

Ключевая особенность: Python придерживается принципа «читаемость важнее краткости». Код на Python понятен даже тем, кто его не писал.
Python 
# Python — простой и читаемый язык
# Пример: вычислить сумму чисел от 1 до 10

сумма = sum(range(1, 11))
print(f"Сумма от 1 до 10 = {сумма}")

# Вывод: Сумма от 1 до 10 = 55

История языка

Python создал Гвидо ван Россум (нидерл. Guido van Rossum) в конце 1980-х годов. Работая в Нидерландах, он хотел создать язык, который был бы мощнее языка ABC, но при этом простым и понятным.

Год Событие
1989 Гвидо ван Россум начинает разработку Python как хобби-проект
1991 Выпущена первая версия Python 0.9.0 — начало публичной истории
1994 Выход Python 1.0 — добавлены lambda, map, filter, reduce
2000 Python 2.0 — добавлена сборка мусора, поддержка Unicode
2008 Python 3.0 — крупное обновление с несовместимостью с Python 2
2020 Официальное завершение поддержки Python 2. Только Python 3
2024–2025 Python стабильно занимает 1-е место по индексу TIOBE
Важно: сегодня используется только Python 3. Python 2 устарел и не поддерживается с января 2020 года. Все примеры в этом курсе написаны на Python 3.

Где применяется Python

Python — один из самых универсальных языков в мире. Его используют в самых разных областях:

Веб-разработка

Такие фреймворки, как Django и Flask, позволяют создавать полноценные веб-приложения. На Python работают крупные сервисы — Instagram, Pinterest, Disqus.

Python — Flask 
from flask import Flask

app = Flask(__name__)

@app.route('/')
def главная():
    return "Привет от Python-сервера!"

# Простейший веб-сервер готов за 7 строк

Анализ данных и машинное обучение

Python — стандарт в Data Science. Библиотеки NumPy, pandas, scikit-learn и TensorFlow сделали его основным языком для специалистов по данным и ML-инженеров.

Автоматизация

Python отлично подходит для автоматизации рутинных задач: работы с файлами, парсинга сайтов, отправки почты, взаимодействия с API. Это экономит часы работы.

Python — Автоматизация 
import os

# Переименовать все .txt файлы в папке
for имя_файла in os.listdir("."):
    if имя_файла.endswith(".txt"):
        новое_имя = "backup_" + имя_файла
        os.rename(имя_файла, новое_имя)
        print(f"Переименовано: {имя_файла} → {новое_имя}")

Научные вычисления

Python используется в NASA, CERN и других научных организациях для моделирования, численных расчётов и визуализации данных.

Преимущества Python

Простой и читаемый синтаксис

Синтаксис Python напоминает псевдокод. Отступы вместо фигурных скобок делают код аккуратным и единообразным. Сравните Python с C++:

Сравнение 
# Python — "Привет, мир!"
print("Привет, мир!")

# C++ — то же самое
# #include <iostream>
# int main() {
#     std::cout << "Привет, мир!" << std::endl;
#     return 0;
# }

Кроссплатформенность

Код Python работает на Windows, macOS, Linux и даже Android без изменений. Достаточно установить интерпретатор Python на любую платформу.

Огромная экосистема библиотек

Репозиторий PyPI содержит более 500 000 пакетов. Для любой задачи скорее всего уже существует готовая библиотека — не нужно писать всё с нуля.

  • pandas — анализ и обработка данных
  • requests — HTTP-запросы к API
  • matplotlib — визуализация данных
  • Pillow — обработка изображений
  • SQLAlchemy — работа с базами данных
  • pytest — автоматизированное тестирование

Большое сообщество

Python — один из самых популярных языков в мире. Это значит, что на любой вопрос найдётся ответ на Stack Overflow, GitHub или в официальной документации. Обучающих материалов на любом языке — тысячи.

Итог: Python — идеальный первый язык программирования благодаря чистому синтаксису, и одновременно мощный инструмент профессионального разработчика. Именно поэтому он занимает первое место по популярности в индексе TIOBE уже несколько лет подряд.
