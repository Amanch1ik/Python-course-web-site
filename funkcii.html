Функции в Python

Функции позволяют разбить программу на переиспользуемые блоки — это основа структурированного кода.

Определение функции

Функция — именованный блок кода, который можно вызвать в любом месте программы. Ключевое слово def объявляет функцию. Тело функции выделяется отступом.

Python 
# Объявление функции
def поприветствовать():
    print("Привет! Добро пожаловать в курс Python.")

# Вызов функции
поприветствовать()   # Привет! Добро пожаловать в курс Python.
поприветствовать()   # можно вызывать сколько угодно раз
Зачем нужны функции: без функций одинаковый код пришлось бы копировать в разных местах. Функции позволяют написать логику один раз и переиспользовать её — это принцип DRY (Don't Repeat Yourself — «не повторяйся»).

Параметры и аргументы

Параметры — это переменные в объявлении функции. Аргументы — конкретные значения, которые передаются при вызове.

Python 
# имя — параметр
def поприветствовать(имя):
    print(f"Привет, {имя}!")

поприветствовать("Нурмухаммед")   # Привет, Нурмухаммед!
поприветствовать("Айгерим")      # Привет, Айгерим!

# Несколько параметров
def сложить(a, b):
    print(a + b)

сложить(3, 5)    # 8
сложить(10, 20)  # 30

Параметры по умолчанию

Python 
def поздороваться(имя, приветствие="Привет"):
    print(f"{приветствие}, {имя}!")

поздороваться("Нурмухаммед")             # Привет, Нурмухаммед!
поздороваться("Нурмухаммед", "Салам")   # Салам, Нурмухаммед!

Возврат значения — return

Оператор return завершает выполнение функции и возвращает значение. Без return функция возвращает None.

Python 
def площадь_прямоугольника(ширина, высота):
    площадь = ширина * высота
    return площадь

результат = площадь_прямоугольника(5, 3)
print(f"Площадь: {результат}")    # Площадь: 15

# Возврат нескольких значений
def мин_макс(числа):
    return min(числа), max(числа)

минимум, максимум = мин_макс([3, 1, 7, 2, 9])
print(минимум, максимум)          # 1 9

Lambda-функции

Lambda — однострочная анонимная функция. Используется там, где нужна простая функция без официального объявления через def.

Python 
# Обычная функция
def удвоить(x):
    return x * 2

# То же самое через lambda
удвоить = lambda x: x * 2
print(удвоить(5))      # 10

# Lambda в sorted() для сортировки по ключу
студенты = [
    {"имя": "Айгерим", "балл": 85},
    {"имя": "Нурмухаммед", "балл": 92},
    {"имя": "Даниял", "балл": 78}
]
по_баллу = sorted(студенты, key=lambda s: s["балл"], reverse=True)
for с in по_баллу:
    print(f"{с['имя']}: {с['балл']}")

Полезные встроенные функции

Python 
числа = [4, 7, 2, 9, 1]

print(len(числа))         # 5       — длина
print(sum(числа))         # 23      — сумма
print(min(числа))         # 1       — минимум
print(max(числа))         # 9       — максимум
print(sorted(числа))      # [1,2,4,7,9] — сортировка
print(abs(-15))           # 15      — модуль
print(round(3.14159, 2))  # 3.14    — округление
print(range(5))           # range(0,5)
print(list(range(5)))     # [0,1,2,3,4]
