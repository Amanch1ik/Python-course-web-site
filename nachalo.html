Начало работы с Python

Установите Python, настройте среду разработки и напишите свою первую программу.

Установка Python

Python можно бесплатно скачать с официального сайта python.org. Установка занимает несколько минут и доступна для всех операционных систем.

Windows

  1. Перейдите на python.org/downloads
  2. Нажмите кнопку Download Python 3.x.x
  3. Запустите скачанный файл .exe
  4. Обязательно поставьте галочку Add Python to PATH
  5. Нажмите Install Now и дождитесь завершения
Важно: галочка Add Python to PATH — обязательна. Без неё Python не будет доступен из командной строки.

macOS

На macOS Python 3 можно установить через официальный установщик с python.org или через менеджер пакетов Homebrew:

Terminal 
brew install python3

Проверка установки

После установки откройте командную строку (Windows) или терминал (macOS/Linux) и введите:

Командная строка 
python --version

Вы должны увидеть что-то вроде:

Вывод
Python 3.12.3
Интерактивный режим: введите просто python в терминале — откроется интерактивная консоль Python (REPL), где можно вводить и сразу выполнять команды. Для выхода введите exit().

Среда разработки (IDE)

Хотя Python-код можно писать в блокноте, для комфортной работы нужен редактор кода с подсветкой синтаксиса, автодополнением и отладчиком.

РедакторУровеньОсобенности
VS Code Новичок / Профи Бесплатный, расширяемый, плагин Python от Microsoft. Рекомендуется.
PyCharm Профи Мощная IDE специально для Python. Есть бесплатная Community-версия.
IDLE Новичок Встроен в Python. Подходит для первых шагов, но функционал ограничен.
Jupyter Notebook Данные / Анализ Идеален для анализа данных — можно запускать код блоками прямо в браузере.

Настройка VS Code для Python

  1. Скачайте VS Code с code.visualstudio.com
  2. Откройте VS Code и перейдите во вкладку Extensions (Ctrl+Shift+X)
  3. Найдите и установите расширение Python от Microsoft
  4. Нажмите Ctrl+Shift+P, введите Python: Select Interpreter
  5. Выберите установленный Python 3

Первая программа

Создайте новый файл с расширением .py — например, main.py — и введите:

main.py 
print("Привет, мир!")

Это классическая программа «Hello, World!» — традиционное первое приложение при изучении любого языка программирования.

Запуск файла

Есть два способа запустить Python-файл:

Способ 1 — через терминал

Командная строка 
python main.py

Способ 2 — в VS Code

Нажмите кнопку Run (треугольник) в правом верхнем углу или клавишу F5.

Вывод
